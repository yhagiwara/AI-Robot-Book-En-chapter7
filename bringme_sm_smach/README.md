# bringme_sm_smach

7.4節のサンプルプログラム  
Bring meタスクのためのSmachによるステートマシンのプログラム


## 実行

- 端末を開いてステートマシーンを実行します．
  ```console
  $ ros2 launch bringme_sm_smach bringme.launch.py
  ```
