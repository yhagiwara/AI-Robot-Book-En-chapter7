# sample_sm_smach

旧7.3節のサンプルプログラム  
Smachによる二状態のステートマシンのプログラム


## 実行

- 端末を開いて二状態のステートマシーンを実行します．
  ```console
  $ ros2 run sample_sm_smach sample_sm_smach
  ```
